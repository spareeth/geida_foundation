# Day 2 — Modules 2 & 3: Spatialising Projects & WaPOR

**Tuesday 7 July 2026 · Hybrid: IsDB HQ Jeddah + Online**

---

## Learning objectives

By the end of Day 2, participants will be able to:

1. Describe how EO contributes at each phase of the IsDB project cycle, from Project Concept Note (PCN) through Project Performance Evaluation Report (PPER), with at least one concrete example per phase
2. Identify relevant EO-derived indicators for at least two sectors and two project phases
3. Generate a sector-specific eToolkit report using a defined project boundary and the appropriate theme filter, and interpret at least two outputs
4. Describe what FAO WaPOR (Water Productivity Open-access Portal) provides, identify which spatial level applies to a given task, and explain when WaPOR is more appropriate than the eToolkit
