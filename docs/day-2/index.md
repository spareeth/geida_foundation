# Day 2 — Modules 2 & 3: Spatializing Projects & WaPOR

**Tuesday 7 July 2026 · Hybrid: IsDB HQ Jeddah + Online**

---

## Learning objectives

By the end of Day 2, you will be able to:

1. Walk through how EO supports each phase of the IsDB project cycle — PCN through PPER — with concrete examples
2. Identify relevant EO indicators for at least two sectors and two project phases
3. Generate a sector-specific eToolkit report using a project boundary and the correct theme filter
4. Describe what FAO WaPOR provides and when to use it instead of eToolkit
