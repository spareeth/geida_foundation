## Day 1 — Facilitator Notes

### Session timing
- Welcoming remarks often run long — protect the 09:30 start for Session 1
- Session 2 (EO Fundamentals) can comfortably extend into the 11:30 slot if discussion is good
- Exercise 1 (15:00–16:00): 1A (Google Earth KML) takes ~20 min; 1B (eToolkit) takes ~35 min; leave 5 min for group share

### Common questions
- "Why can't I use Google Earth desktop?" — web version is sufficient for drawing KML; desktop version is more powerful but requires installation
- "My eToolkit analysis is taking too long" — area is probably too large; ask participant to reduce the polygon
- "The KML won't import into eToolkit" — draw directly on the eToolkit map instead; refer to the on-screen drawing tools

### Online participant tips
- Ask online participants to turn on cameras for the round of introductions
- Exercise 1A works identically for online participants in their own browser
- Breakout rooms useful for small-group discussion during Session 2

## Day 2 — Facilitator Notes

### Session timing
- The morning session (PCN–PCR walkthrough) is dense — watch for participant fatigue after 10:30
- Official opening is scheduled 13:30–14:30 but senior speakers may run short or long — build in buffer
- WaPOR exercise (15:30–16:30): some participants need help finding the country and time-series UI; circulate actively

### Common questions
- "Which version of the slide deck should I use?" — v1 (demo area) for the lecture, then show v2 and v3 as examples
- "Why are there three versions of the same deck?" — to show the methodology works across sectors

---

## Day 3 — Facilitator Notes

### Session timing
- GeoLibre demo (afternoon): allow 20–30 min for the GADM download + GeoLibre loading workflow before participants try it themselves
- HDX is less structured than GADM — participants often need help finding their country's spatial data; set expectations

### Common questions
- "The GADM file won't open in GeoLibre" — they may not have unzipped it first; the Shapefile must be inside a ZIP for GeoLibre to read it
- "I can't find my district in the GeoJSON" — they need to use GeoLibre's selection tool; demo this step explicitly

---

## Day 4 — Facilitator Notes

### Case study facilitation
- Groups of 3–4 work best; mix departments so field-facing and HQ staff are in each group
- Allow 15 min at the start of each case study for groups to agree on which project to use
- Circulate during group work to catch groups who are stuck on the tool (not on the analysis)

### Presentation feedback
- Keep feedback brief and constructive — 2–3 points per group maximum
- Emphasise what was done well before suggesting improvements
- Key points to highlight where relevant: legend positioning, data source citation, connecting EO outputs to specific project documents

### Certificate distribution
- Confirm participant list against attendance log before distributing certificates
- Online participants who completed all four days receive certificates by email
